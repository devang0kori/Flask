About This Application

Hello! I am currently learning Flask, and this is a basic Todo app that I built as part of my learning journey.

This application allows you to perform the following CRUD (Create, Read, Update, Delete) operations:

  • Create: Add new todo items.
  • Read: View all existing todo items.
  • Update: Edit the details of an existing todo item.
  • Delete: Remove a todo item that you no longer need.

Thank you for checking out my project!
